About Focus Insight

Focus Insight는 데모용으로 제작된 제출용 웹입니다. 실제 측정은 아니며, CPT/AX-CPT 같은 검사의 개념을 바탕으로 한 프로파일 시각화를 보여줍니다.

Method (Demo)

입력 텍스트 기반의 임의 점수 산정, 3개 지표(집중력·억제능력·지속성)를 시각화하여 데모 리포트를 제공합니다. 이후 Gemini API 연동 시 모델 기반 분석으로 확장 가능합니다.
